# StoryBook

Imaginary blog where users can post their short stories

## Getting Started

```shell
cd Storybook
yarn
yarn run start
```

### Prerequisites

-   `node`
-   `npm || yarn`

## Running the tests

> WIP

`yarn run test` || `npm run test`

## Deployment

Deployed to Heroku, but cant be used there because of CORS.
